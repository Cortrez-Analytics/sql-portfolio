**Airline Delay Analysis (SQL Project)**

**Objective**
Analyze airline delay performance to identify key trends, high-delay airports, and primary causes of delays.

**Dataset** 
Kaggle airline delay dataset containing aggregated flight performance data by airport and carrier.

**Key Questions** 
- Which airports have the highest delay rates?
- Which carriers perform the worst?
- What are the main causes of delays?
- How do delays vary over time?

**Key Insights** 
- Late aircraft and weather were the largest contributors to delays  
- Certain airports consistently showed higher delay rates  
- Delay rates varied significantly across carriers  

**SQL Skills Used** 
- GROUP BY and aggregations (SUM, AVG, COUNT)  
- KPI calculation (delay rate)  
- Data filtering and sorting  
- Basic trend analysis
